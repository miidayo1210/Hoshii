# Hoshii プロジェクト バックアップ
作成日時: Thu Oct 23 07:27:05 JST 2025

## 含まれるファイル
- app/: Next.jsアプリケーション
- components/: Reactコンポーネント
- lib/: ライブラリとユーティリティ
- supabase/: データベーススキーマ
- .env.local: Supabase環境変数
- package.json: 依存関係

## 復元方法
1. 新しいプロジェクトディレクトリを作成
2. このバックアップの内容をコピー
3. npm install を実行
4. npm run dev:local で起動
